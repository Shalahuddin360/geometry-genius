3. What is event bubble?

Event Bubble : The "Event Bubbling" behavior makes it possible for you to handle an event in a parent element instead of the actual element that received the event.The pattern of handling an event on an ancestor element is called Event Delegation

4.What is a callback function and why will you use it?

Callback:A callback function is a function passed as an argument of another function, which will be executed eventually. The callback function executes inside the main function and it is up to the main function to decide when to execute it

why will you use it: Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors
